# DBE teacher workshop – Updated Scratch Curriculum (Days 1–2)

These folders are the **Scratch** portion of the Department of Basic Education (DBE) coding and robotics **teacher training** curriculum (week-long workshop).

| Folder | Content |
|--------|--------|
| **day-01-scratch** | Introduction to Scratch, lesson plan, activities, assessment, **project templates** (moving cat, story, score keeper with timer, number guessing game, etc.) |
| **day-02-scratch** | Advanced Scratch, lesson plan, **activities with example block scripts** (cloning, broadcasting, catch game, list quiz), assessment |

**Upstream / updates:** [github.com/msovara/dbe-coding-robotics-curriculum](https://github.com/msovara/dbe-coding-robotics-curriculum)  

**CHPC DSI Coding School:** [github.com/ChpcTraining/dsi_coding_school](https://github.com/ChpcTraining/dsi_coding_school)
